### ğŸ’¡ Responsiveness Breakpoints

The design uses the following standard breakpoints defined in `styles.css`:

1.  **Mobile-First (Default):** `(max-width: 576px)`
2.  **Tablet:** `(min-width: 577px) and (max-width: 998px)`
3.  **Desktop:** `(min-width: 999px)`

### ğŸ“ Key Design Notes

  * **Color Palette:** The design utilizes a warm, professional palette featuring primary colors like **Orange (`#F58420`)** and **Brown (`#462512`)** against a **White/Off-White (`#FFF9F4`)** background.
